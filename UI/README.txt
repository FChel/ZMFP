Object...........: ZFSS_SSP
Object Type......: SAP UI5 Application
Title............: Self Service Purchasing (SSP) App
Description......: The SSP App or Self-Service Purchasing MyFi Application is being developed to process
                   ROMAN Outline Agreements (OA) and Purchase Orders (PO). The app will replace part of
                   the functionality currently undertaken via the AE643 Defence Purchasing web-form.

                   SSP provides an opportunity and technology platform to extend the finance self-service
                   offering through a modern user interface, real-time validation and improved compliance.
                   Release 1 of SSP to a limited user group is scheduled for early 2020, with Release 2
                   planned Defence-wide in mid 2020.

       Release 1 of the App will roll out with the following functionality:
       - Create an OA only
       - Create OA with a single PO (like for like)
       - Change an OA created via the app
       - Change an OA with single PO (like-for-like) that was created via the app
       - Search and display OAs that were created via the app
